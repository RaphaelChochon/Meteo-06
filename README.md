Météo06
=======

## Préambule
Ce site a été développé dans le but de créer un point d'accès (fiable, fluide et moderne) aux données des stations météos de notre réseau. Toutes les stations de notre réseau sont équipées d'un Raspberry Pi sur lesquels est installé [Weewx](https://github.com/weewx/weewx), un logiciel permettant de se connecter à la console de la station et ainsi de récupérer les données.
